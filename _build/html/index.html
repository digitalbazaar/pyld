Welcome to PyLD!¶

PyLD is a Python implementation of a JSON-LD processor.

pyld.jsonld.compact(input_, ctx, options=None)[source]¶

Performs JSON-LD compaction.

Parameters:

  • input – the JSON-LD input to compact.

  • ctx – the JSON-LD context to compact with.

  • [options] –

    the options to use. [base] the base IRI to use. [compactArrays] True to compact arrays to single values when

    appropriate, False not to (default: True).

    [graph] True to always output a top-level graph (default: False). [expandContext] a context to expand with. [extractAllScripts] True to extract all JSON-LD script elements

    from HTML, False to extract just the first (default: False).

    [processingMode] Either ‘json-ld-1.0’ or ‘json-ld-1.1’,

    defaults to ‘json-ld-1.1’.

    [documentLoader(url, options)] the document loader

    (default: _default_document_loader).

Returns:

the compacted JSON-LD output.

pyld.jsonld.normalize(input_, options=None)[source]¶

Performs RDF dataset normalization on the given input. The input is JSON-LD unless the ‘inputFormat’ option is used. The output is an RDF dataset unless the ‘format’ option is used’.

Parameters:

  • input – the JSON-LD input to normalize.

  • [options] –

    the options to use. [algorithm] the algorithm to use: URDNA2015 or URGNA2012

    (default: URGNA2012).

    [base] the base IRI to use. [inputFormat] the format if input is not JSON-LD:

    ’application/n-quads’ for N-Quads.

    [format] the format if output is a string:

    ’application/n-quads’ for N-Quads.

    [extractAllScripts] True to extract all JSON-LD script elements

    from HTML, False to extract just the first (default: False).

    [processingMode] Either ‘json-ld-1.0’ or ‘json-ld-1.1’,

    defaults to ‘json-ld-1.1’.

    [documentLoader(url, options)] the document loader

    (default: _default_document_loader).

Returns:

the normalized output.

class pyld.DocumentLoader[source]¶

Abstract base class for class-based JSON-LD document loaders.

Concrete subclasses implement __call__() to fetch a document for a given URL and return a RemoteDocument.

PyLD’s loader contract remains “any callable with the right signature”. This ABC documents the interface used by built-in class-based loaders.

class pyld.FrozenDocumentLoader(documents: Mapping[str, dict | Path] | None = None)[source]¶

Document loader that serves only a sealed allowlist of URLs.

documents maps each allowed URL to either a parsed JSON-LD dict or a pathlib.Path pointing to a JSON file on disk. Path entries are read and parsed lazily on first request, then cached in place so subsequent calls skip the file read. Any URL not present in the mapping raises pyld.jsonld.JsonLdError with code 'loading document failed'.

With no arguments, a FrozenDocumentLoader serves the curated BUNDLED_CONTEXTS set. To extend rather than replace the bundle:

FrozenDocumentLoader(documents=dict(BUNDLED_CONTEXTS, **extras))
