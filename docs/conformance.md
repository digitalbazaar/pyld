# :fontawesome-solid-legal: Conformance

PyLD aims to conform with:

- [JSON-LD 1.1](https://www.w3.org/TR/json-ld11/)
- [JSON-LD 1.1 Processing Algorithms and API](https://www.w3.org/TR/json-ld11-api/)
- [JSON-LD 1.1 Framing](https://www.w3.org/TR/json-ld11-framing/)
- [RDF Dataset Canonicalization (URDNA2015)](https://www.w3.org/TR/rdf-canon/)
- [RDF Dataset Normalization (URGNA2012)](https://www.w3.org/TR/rdf-graph-normalization/)
- The JSON-LD Working Group [test suite](https://github.com/w3c/json-ld-api/tree/master/tests)

The test runner is updated over time to note or skip newer tests that are not
yet supported.
