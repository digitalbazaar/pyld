    Conformance¶

    PyLD aims to conform with:

    • JSON-LD 1.1
    • JSON-LD 1.1 Processing Algorithms and API
    • JSON-LD 1.1 Framing
    • Universal RDF Dataset Canonicalization Algorithm 2015 (URDNA2015)
    • Universal RDF Graph Canonicalization Algorithm 2012 (URGNA2012)
    • The JSON-LD Working Group test suite

    PyLD aims to follow JSON-LD Best Practices.

    The test runner is updated over time to note or skip newer tests that are not yet supported.
